Portfolio

- Coding in space -


I was the first web developer to code and build a website while traveling to space. I was approached by NASA to build their new website and I thought it was a great idea to utilise one of their space ships to code in. The website was a success and so was my trip.

- A Poodle of Code -


A website constructed to help Poodles learn how to best take care of a curly fur. Too long has Poodle's been neglected when it comes to self grooming but no more. Learn from famouse and succesfull Poodle's how to take care of those beatiful curly locks.

- Captain Code -


An interactive website application for steam boat captains all over the world where they can plan routes, play thematic games or chat on the revolutionary new forum "full steam ahead".
